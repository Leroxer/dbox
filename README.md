    The dbox project allows all Linux distributions to utilize any package manager.

    Supported package managers:

     apt  
     apt-get  
     apk  
     dnf  
     pacman  
     yay  
     zypper  
     xbps

Recommended desktop environments (those using a common shortcut system):

GNOME  
KDE Plasma  
XFCE  
LXQt  
MATE  
Cinnamon  
Budgie  


## Installation

```sh
sudo curl -o /usr/local/bin/dbox https://raw.githubusercontent.com/leroxer/dbox/dbox
sudo chmod +x /usr/local/bin/dbox


## Prepare Containers

```sh
dbox init         # Run this once to set up desktop integration
dbox desktop-sync
